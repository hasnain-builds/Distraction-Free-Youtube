# 🎯 Distraction-Free YouTube

A simple web project built using **HTML** that provides a distraction-free YouTube experience by removing unnecessary elements like recommendations, comments, and sidebars.

## 🚀 Features

- Clean and minimal UI
- Focus only on video content
- Removes distractions like:
  - Suggested videos
  - Comments section
  - Sidebar content
- Lightweight and fast (pure HTML)

## 🛠️ Tech Stack

- HTML5

## 💡 How It Works

This project simplifies the YouTube interface by:
- Embedding videos using iframe
- Hiding distracting elements using custom layout 
- Keeping the interface minimal for better focus
